This folder's content is meant to be dragged inside the "shaders_slang" folder within Retroarch shader directory. The entire folder should not be dragged inside the folder, but the content inside should.

This shader pack in Slang format is the main focus on the project to get more frequent updates since it's widely supported from many video drivers inside Retroarch such as GLCore, DX11, DX12, Vulkan, and Metal.
